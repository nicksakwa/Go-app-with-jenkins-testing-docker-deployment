# Go Programming Language 


## For this repository, the prerequisites are:

Before diving into this repository, ensure you have the following installed:

1.  **Go vs code extension:** Which you can add from VS Code extensions.
2.  **Go:** Install the library using go mod in your bash terminal or your directory:
    ```bash
    go mod init go-simple-app
    ```


## How to Run Examples

To execute the Go program in this repository:

1.  Navigate to the directory containing the .go files in your terminal.
2.  Run the desired script using the Python interpreter:
    ```bash
    go run main.go "Local User"
    ```
